Getting started


Get started by installing WCS in your project :


npm install wcs-core
yarn add wcs-core


We highly recommend to use the Template guidelines.

Get a full step-by-step installation guide in the Getting Started section.


Contribute

You can contribute by adding new issues for bugs or new features on the official WCS GitLab repository.

Feel free to submit a merge request that will be reviewed by our team.
