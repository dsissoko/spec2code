Design Tokens


This page is a documentation of the design tokens used in the design system.
Every token found here is represented by a CSS variable prefixed with --wcs.


E.g: primitive.color.parme.500 corresponds to the CSS variable var(--wcs-primitive-color-parme-500).




If you want to integrate these tokens into your project, please refer to the Theming documentation.
